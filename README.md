# Bestiary:
## Erbide:
  - Genders: male/female
  - Food: leaves
  - Can transport pollen
  - Can reproduce after 3 decaticks of life, 8 ticks cycle
  - Lives for a cycle (centatick) circa
## Solaria:
  - Genders: None
  - Food: None (sun)
  - Born from mushrooms
  - Can't reproduce leaves starting from dead leaves
  - Lives forever
## Funghinoide:
  - Genders: None
  - Food: None
  - Born from corpses/other mushrooms
  - Extremely short life
## Carnide:
  - Genders: male/female
  - Food: Erbides
  - Actively hunts nearby prey when hungry
  - Can reproduce after 3 decaticks of life, 8 ticks cycle
  - Lives for about one cycle (centatick)
  - Starves if it cannot hunt
  - Performs short ambush bursts before attacking prey
  - Usually gives birth to 0–3 offspring (each with 50% chance)
## Terminaria (WIP)
  - Genders: None
  - Food: spores
  - Carnivorous plant
  - Grows a terminal leaf used to lure herbivores
  - Kills nearby herbivores to feed the Symplants growing at its roots
  - Cannot move
  - Dies if deprived of spores
